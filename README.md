# kalos.partners

Static landing page for Kalos Partners.

Deployed via Cloudflare Pages from this repo.

## Files

- `index.html` — landing page
- `privacy.html` — privacy policy (GDPR-compliant)

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static HTTP server.

## Deployment

Connected to Cloudflare Pages. Pushes to `main` deploy automatically to `kalos.partners`.
