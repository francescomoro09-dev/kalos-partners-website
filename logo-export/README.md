# Kalos Partners — Logo PNGs

PNG exports of the Kalos Partners "K" mark for use in email avatars and other places that need a raster image.

## Files

| File | Size | Best for |
|---|---|---|
| `kalos-logo-256.png` | 256 × 256 | Slack, small avatars |
| `kalos-logo-512.png` | 512 × 512 | **Email profile picture (recommended)** — Google Workspace, Outlook, iCloud all accept this size |
| `kalos-logo-1024.png` | 1024 × 1024 | High-res / print / future use |

## Notes

- Background: `#1f2a24` (dark forest green — same as site)
- Letter colour: `#ede8d8` (warm cream — same as site)
- The K is set in Georgia serif. The icon has rounded corners — when an email client crops it to a circle, the corners are clipped but the letter stays fully visible.
- The master vector is `../favicon.svg`.

## How to set as your email profile picture

**Google Workspace / Gmail:** [myaccount.google.com](https://myaccount.google.com) → Personal info → Photo → upload `kalos-logo-512.png`.

**Outlook:** Profile picture in top-right of Outlook → upload the file.
